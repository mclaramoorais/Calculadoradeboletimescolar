Boletim Escolar: 
Aplicação web simples em HTML, CSS e JavaScript para simular o cálculo de um boletim escolar.

Funcionalidades:
Inserção de 4 notas
Cálculo da média inicial
Verificação de aprovação ou recuperação
Substituição da menor nota pela recuperação (se maior)
Avaliação final, se necessário
Exibição de resumo completo

Tecnologias:
HTML
CSS
JavaScript

Como executar:
Abra o arquivo index.html no navegador
Preencha as notas
Clique em Gerar resumo

Observação:
Notas devem estar entre 0 e 10.
